Create client/src/lib/workspace.jsx implementing:

1. WorkspaceContext with default shape:
   `{ workspaces: [], activeWorkspaceId: null, activeWorkspace: null, setActiveWorkspace: () => {}, loading: true }`

2. WorkspaceProvider component:
   - Reads `user` from `useAuth()`; when user is null, provides empty context and skips loading
   - On user load, queries Supabase: `supabase.from('workspace_users').select('workspace_id, workspaces(id, name, invite_code)').eq('user_id', user.id)`
   - Maps result to array of workspace objects: `{ id, name, invite_code }`
   - Resolves activeWorkspaceId using this priority:
     a. Value from `localStorage.getItem('activeWorkspaceId')` if it matches a workspace in the list
     b. Falls back to `workspaces[0].id` if stored value is invalid/missing
     c. null if user has no workspaces
   - `setActiveWorkspace(id)` updates state AND writes to `localStorage.setItem('activeWorkspaceId', id)`
   - Derives `activeWorkspace` as the workspace object matching `activeWorkspaceId`
   - Provides `{ workspaces, activeWorkspaceId, activeWorkspace, setActiveWorkspace, loading }` via context

3. `useWorkspace()` hook — throws if used outside WorkspaceProvider (same guard pattern as useAuth)

Export: `WorkspaceProvider`, `useWorkspace`
