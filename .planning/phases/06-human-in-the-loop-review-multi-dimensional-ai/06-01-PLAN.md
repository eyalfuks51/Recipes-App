  'ישראלי','מרוקאי','עיראקי','תוניסאי','צרפתי',"פיוז'ן",'אחר',

  'עתיר חלבון','דל פחמימה','מושחת','קליל',

Replace HEBREW_SYSTEM_PROMPT with an expanded version that:
1. Opens with explicit Hebrew instruction: "חשוב ביותר: כל הטקסט בתשובה חייב להיות בעברית בלבד."
2. Instructs the AI to return a single JSON object with ALL of the following fields:
   - `title` (string, Hebrew)
   - `main_category` (string, exactly one of ALLOWED_CATEGORIES list)
   - `difficulty` (string, one of: קל, בינוני, קשה)
   - `ingredients` (array of Hebrew ingredient name strings)
   - `meal_type` (string, e.g. ארוחת בוקר, ארוחת ערב, חטיף, קינוח — free-form Hebrew)
   - `cuisine` (string, exactly one of ALLOWED_CUISINES list)
   - `main_ingredient` (string, single primary ingredient in Hebrew)
   - `equipment_needed` (array of Hebrew equipment strings, [] if none)
   - `prep_time` (integer minutes or null if not determinable)
   - `cook_time` (integer minutes or null if not determinable)
   - `dietary_tags` (array, each element must be one of ALLOWED_DIETARY_TAGS, [] if none)
   - `instructions` (array of Hebrew step strings — no number prefix in each string, [] if no steps)
3. Instructs: "אל תוסיף טקסט מחוץ ל-JSON."

After parsing, add normalization after the existing main_category normalization:
```javascript
// Normalize cuisine
if (!ALLOWED_CUISINES.includes(recipe.cuisine)) {
  recipe.cuisine = 'אחר';
}
// Normalize dietary_tags — keep only recognized values
if (!Array.isArray(recipe.dietary_tags)) {
  recipe.dietary_tags = [];
} else {
  recipe.dietary_tags = recipe.dietary_tags.filter(tag => ALLOWED_DIETARY_TAGS.includes(tag));
}
// Normalize arrays that may be missing
if (!Array.isArray(recipe.instructions)) recipe.instructions = [];
if (!Array.isArray(recipe.equipment_needed)) recipe.equipment_needed = [];
```

Keep all existing code unchanged (createClient, test hook, error handling, regex JSON extraction).
