Insert the following normalization block immediately after those 3 lines (before the `// Normalize category` comment):

```javascript
// Normalize ingredient shape: AI should return [{name, amount, unit}] objects.
// If AI returns plain strings (legacy or fallback), convert to object shape.
recipe.ingredients = recipe.ingredients.map(item => {
  if (typeof item === 'string') {
    return { name: item.toLowerCase().trim(), amount: null, unit: null };
  }
  return {
    name: (item.name ?? '').toLowerCase().trim(),
    amount: item.amount ?? null,
    unit: item.unit ?? null,
  };
}).filter(item => item.name.length > 0);
```

Do not change any other code in the function. The validation line above this block stays in place.
