  <action>
Create server/Dockerfile:

```
FROM node:20-alpine

WORKDIR /app

COPY package*.json ./
RUN npm ci --omit=dev

COPY . .

EXPOSE 3000

CMD ["node", "src/index.js"]
```

Notes:
- Use node:20-alpine (smallest stable LTS)
- npm ci --omit=dev skips devDependencies (correct practice even if none exist)
- No build step needed — pure ESM source runs directly
- EXPOSE 3000 is documentation only; Koyeb reads PORT env var at runtime

Create server/.dockerignore:

```
node_modules
.env
.env.*
!.env.example
npm-debug.log*
```

Notes:
- Exclude node_modules (rebuilt inside image via npm ci)
